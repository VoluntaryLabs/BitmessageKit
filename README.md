BitmessageKit
==============

An objective-c framework you can use to add Bitmessage support to your apps.


Dependencies
--------------------

These are included in the BitmessageKit/Vendor folder:

* pybitmessage
* static-python
* tor
* xlmrpc
* ZipKit
